          SNAPKEY AGENT - SUPPORT TEAM INSTRUCTIONS

🎉 EASY SETUP FOR NON-TECHNICAL USERS 🎉

Just follow these simple steps:

1. **DOWNLOAD THE FILES**
   You will receive a ZIP file containing:
   - `agent.exe` (the main program)
   - `SnapKey-Agent-Starter.bat` (easy startup)
   - `README-FOR-SUPPORT-TEAM.txt` (this file)

2. **INSTALLATION**
   - Unzip the files to any folder (Desktop is fine)
   - No installation needed! Just double-click to run.

3. **CONFIGURATION**
   Open `SnapKey-Agent-Starter.bat` in Notepad and edit these lines:

   set BACKEND_WS_URL=ws://your-backend-url.com/ws/agent
   set AGENT_TOKEN=your-unique-team-token

   (Your team leader will provide the correct values)

4. **RUNNING THE AGENT**
   - Double-click `SnapKey-Agent-Starter.bat`
   - You'll see a window showing the agent is running
   - Keep this window open while working

5. **AUTOMATIC STARTUP (Optional)**
   To run automatically when Windows starts:
   - Press Win+R, type `shell:startup`, click OK
   - Copy the `SnapKey-Agent-Starter.bat` file into this folder

=================================================================
                    REQUIREMENTS
=================================================================

✅ Windows 10 or 11
✅ AnyDesk installed (for remote connections)
✅ Internet connection
✅ That's it! No technical skills needed.

=================================================================
                    TROUBLESHOOTING
=================================================================

🔴 "Cannot connect to backend"
   - Check your internet connection
   - Verify the backend URL in the batch file

🔴 "Invalid agent token"
   - Contact your team leader for the correct token

🔴 "AnyDesk not found"
   - Install AnyDesk from https://anydesk.com/

🟢 "Connected successfully"
   - Agent is working! You're ready to receive support requests.
